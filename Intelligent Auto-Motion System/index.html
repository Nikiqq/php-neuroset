Информация о программе

Приветствуем вас в нашей программе моделирования дорожного движения на перекрестке. В данной программе реализовано 4 режима работы: 2 режима для красного светофора и 2 для зеленого. На каждый режим светофоров влиет набор входных переменных, который определяется так:

  • Время суток (X)
  • Число машин перед стоп-линией на красном свете для главной дороги (Y)
  • Количество пешеходов с 2 сторон главной дороги (Z)
.

X = 0 при t ϵ (22:00, 08:00) ᴗ (10:00, 18:00)
X = 1 при t ϵ [08:00, 10:00] ᴗ [18:00, 22:00]

Y = 0 при N < = 7 машин
Y = 1 при N > 7 машин

Z = 0 при M < = 40 человек
Z = 1 при M > 40 человек


В результате опытных исследований получились следующие связи входных и выходных переменных:

GL = Y*(!(X+Z) + X) - продолжительность работы зеленого светофора
RL = Z*((!X*Y) + X) - продолжительность работы красного светофора


Входные переменные можете задавать и вы, для этого вам нужно поменять количество машин на всем перекрестке, или сменить время на таймере (с нажатием enter).
